library(dplyr)
library(ggplot2)
file <- read.csv("AnoAtual.csv")

frameEmpresaAerea <- as.data.frame(table(empresaAerea))

  theme(axis.text.x = element_text(angle = 45, hjust = 1), panel.background=element_blank(),

  scale_x_discrete(breaks = 1:5)

Ao analisar o documento de cota para exercicio da ativdade parlamentar, possivel notar que algumas despesas so negativas ou tem o valor igual a zero. O que em outros casos poderia significar algum erro nos dados, nesse caso especifico, isso acontece pois se trata de um bilhete de compensao. Significa que foi emitido um bilhete e no foi utilizado pelo deputado. O caso de despesa negativa ocorre para a emisso de bilhete areo. J o caso de despesa igual a zero ocorre quando o tipo telefonia, o que significa que a despesa foi franqueada.

J que apenas um tipo de despesa pode ter valor negativo o mais indicado que se tenha maior atenao para esse tipo de depensa, pois algumas metricas sero prejudicadas, como por exemplo, media, desvio padro, variancia.
